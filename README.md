## Paper

### 1. Grove: Boosting 기반 CAT 모델 개발 [리뷰중]

- 참여 인원: 3명
- 문제 정의 및 연구 목표
  - : 기존 CAT은 IRT 기반 가정(단일 차원성, 지역 독립성)에 의존하여 실제 데이터 적용에 한계 존재
  - : 부스팅 모델의 높은 예측력을 활용하면서 CAT의 적응적 문항 선택 구조를 구현하기 위한 Grove 알고리즘 제안
- 데이터 및 평가 지표
  - : VIQ(45문항, 11,502명), TMAS(50문항), KMLE(360문항) 데이터 사용
  - : ROC-AUC, Accuracy, 응시 문항 수, 항목 노출 분산, 과적합 여부
- 주요 성과
  - : 부스팅 모델을 개별 트리로 분해하고 서브그룹으로 재구성한 Grove 알고리즘 설계 및 구현
  - : DT 기반 CAT 대비 높은 ROC-AUC 및 유사한 Accuracy 달성, CatBoost 성능에 근접
  - : 트리 조합을 통한 다양한 테스트 폼 생성 및 항목 노출 분산 효과 확인
- 세부 설명: [Grove](https://github.com/novicedata/Paper/tree/main/Grove_CAT)

### 2. MZ세대 공무원 이직의도 분석: RuleFit 기반 설명가능 인공지능(XAI) 연구

- 참여 인원: 3명
- 문제 정의 및 연구 목표
  - : 청년·저연차 공무원의 조기 이탈이 증가하면서 조직 지속가능성과 행정서비스 안정성에 대한 리스크가 확대
  - : 기존 회귀 기반 연구의 한계를 보완하기 위해 RuleFit 기반 설명가능 인공지능(XAI)을 활용하여 MZ세대 공무원의 이직의도에 영향을 미치는 다요인 상호작용 규칙을 탐색
- 데이터 및 평가 지표
  - : 2024 공직생활실태조사 데이터 활용 (MZ세대 공무원 대상 분석)
  - : ROC-AUC, Accuracy 기반 예측 성능 비교
- 주요 성과
  - : RuleFit을 적용하여 직무만족, 보상 인식, 개인 특성 등이 결합된 이직의도 규칙(If-Then rule) 도출
  - : 기존 로지스틱 회귀 대비 높은 예측 성능을 확인하고, 정책 설계에 활용 가능한 해석 가능한 규칙 기반 분석 제시
  - : 특정 조건을 가진 위험 집단을 식별하여 맞춤형 인사·조직 관리 전략 설계 가능성 제시
- 세부 설명: [MZ Turnover](https://github.com/novicedata/Paper/tree/main/MZ_turnover)
